language_id_score_filter#

Filter to keep samples in a specific language with a confidence score above a threshold.

This operator uses a FastText model to identify the language of each sample. It keeps samples that are in the specified language(s) and have a language identification confidence score greater than or equal to the minimum score. If no specific language is provided, it only filters based on the confidence score.
